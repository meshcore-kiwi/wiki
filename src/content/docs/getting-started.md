[MeshCore](https://meshcore.io/) is open-source firmware for off-grid LoRa mesh networking - send messages between low-cost radio devices with no internet, no cell coverage, and no subscription.

## What you need

1. A supported LoRa device - see [recommended devices](/hardware/recommended-devices/)
2. The MeshCore companion app on your phone
3. To be in range of another node (or a mate with one!)

## Radio settings

Use the **New Zealand (Narrow)** preset when setting up any node, repeater, or room server - the whole NZ mesh runs on it, and a node on different settings can't talk to anyone. It's available as a preset in the companion app, or set manually:

| Setting | Value |
| --- | --- |
| Frequency | 917.375 MHz |
| Bandwidth | 62.5 kHz |
| Spreading factor | SF7 |
| Coding rate | CR 5 |
